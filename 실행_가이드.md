# 프로그램 배포 관리 - 실행 가이드

## 빠른 시작 (다른 PC에서 실행 + 데이터 공유 + 외부 URL)

### 1단계: 데이터 공유 설정 (필수)

여러 PC에서 같은 데이터를 쓰려면 **Supabase** 설정이 필요합니다.

- **[SUPABASE_데이터공유_설정.md](SUPABASE_데이터공유_설정.md)** 파일을 열고 단계대로 진행
- `backend/.env` 에 `SUPABASE_URL`, `SUPABASE_KEY` 입력
- **같은 값을 모든 PC에 설정** → 데이터 자동 공유

### 2단계: 실행

| 방법 | 명령 | 비고 |
|------|------|------|
| **원클릭** | `run-public.bat` 더블클릭 | 빌드 + 서버 시작 |
| **Docker** | `docker-compose up -d` | Docker 설치 필요 |

### 3단계: 접속 URL

| 대상 | URL |
|------|-----|
| 이 PC | http://localhost:8000 |
| 같은 네트워크 (다른 PC/휴대폰) | http://[이 PC의 IP]:8000 |
| **인터넷 (외부)** | `run-with-tunnel.bat` 실행 후 표시되는 URL |

### 외부(인터넷) URL

| 방식 | 배치 파일 | URL 특성 |
|------|-----------|----------|
| **고정 URL** (추천) | `run-with-ngrok.bat` | 항상 같은 주소 |
| 임시 URL | `run-with-tunnel.bat` | 실행할 때마다 변경 |

**고정 URL 설정** (ngrok):
1. https://ngrok.com 가입
2. Domains에서 무료 도메인 발급
3. `ngrok-config.txt` 에 도메인 입력
4. `run-with-ngrok.bat` 실행
5. 자세한 설정: **고정URL_설정_가이드.md**

---

## 다른 PC에 설치하기

1. 이 프로젝트 폴더를 **복사** 또는 **git clone**
2. `backend/.env` 에 위 1단계에서 확인한 Supabase URL/KEY 입력 (동일한 값)
3. `run-public.bat` 실행
4. 해당 PC의 IP로 접속: `http://[그 PC의 IP]:8000`

→ **모든 PC가 같은 Supabase를 사용하므로 입력/수정/삭제가 실시간으로 공유됩니다.**
