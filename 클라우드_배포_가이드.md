# 클라우드 배포 - 고정 URL 받기

한 번 배포하면 **고정 접속 URL**을 받을 수 있습니다.  
(PC를 켜두지 않아도, 어디서든 접속 가능)

---

## 접속 URL (배포 후)

| 구분 | URL |
|------|-----|
| **Render 배포** | `https://deploy-manage-XXXX.onrender.com` |
| ※ XXXX | 배포 완료 시 Render 대시보드에 표시되는 실제 값 |

---

## 빠른 배포 (deploy-to-render.html)

1. **deploy-to-render.html** 더블클릭 → 브라우저에서 열기
2. GitHub 저장소 URL 입력
3. **Render에서 배포하기** 클릭
4. Render 가입 후 Apply → 5~10분 후 고정 URL 발급

자세한 순서: **클라우드_배포_완료_URL.md** 참고

---

## Render.com 수동 배포

1. https://render.com 가입
2. **New +** → **Web Service**
3. 저장소 연결 후 이 프로젝트 선택
4. **Runtime**: Docker
5. **Create Web Service**
6. 배포 완료 후 상단 URL 확인

### 환경변수 (데이터 공유)
- **Settings** → **Environment**: `SUPABASE_URL`, `SUPABASE_KEY` 추가

---

## run-all.bat (로컬 실행 + URL 표시)

클라우드 대신 본인 PC에서 실행할 때:

1. **run-all.bat** 더블클릭
2. 빌드 → 서버 시작
3. 화면에 표시되는 **실제 접속 URL** 사용:
   - 이 PC: `http://localhost:8000`
   - 같은 네트워크: `http://[표시된 IP]:8000`
   - 외부(cloudflared 있을 때): `https://xxx.trycloudflare.com`
