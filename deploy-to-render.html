프로그램 배포 관리 - 클라우드 배포

Render.com 에 한 번 배포하면 고정 URL을 받습니다.

1. GitHub에 프로젝트 푸시

이 폴더를 GitHub 저장소에 올리세요.

2. 아래에 저장소 URL 입력 후 배포


Render에서 배포하기

3. 배포 후 접속 URL

https://deploy-manage-XXXX.onrender.com

배포 완료 시 Render 대시보드에 위와 같은 형태의 고정 URL이 표시됩니다.

※ 무료 플랜은 15분 미사용 시 슬립 → 최초 접속 시 1분 내 재시작

4. 데이터 공유 (Supabase)

Render 대시보드 → 서비스 선택 → Environment 에 SUPABASE_URL, SUPABASE_KEY 추가
